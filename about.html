Tools Used

  • Leaflet API was used to show the map of the St. Paul area
  • Nominatim API was used to allow users to input an adress or a latitude and longitude to move the map to the location if the location is in St. Paul
  • A newly custom created St. Paul Crime API retreves the crime data for the most recent crimes
  • The Vue.js framework was used to help build the app and make it reactive
  • The Foundation CSS framework was used to help style the app, including this page
  • The IDEs Visual Studio Code and IntelliJ IDEA was used to edit code
  • Node.js is used to host the St. Paul crime API
