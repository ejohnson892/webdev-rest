Tools Used

  • Leaflet API was used to show the map of the St. Paul area
  • Nominatim API was used to allow users to input an adress or a latitude and longitude to move the map to the location if the location is in St. Paul
  • A newly custom created St. Paul Crime API retreves the crime data for the most recent crimes
  • The Vue.js framework was used to help build the app and make it reactive
  • The Foundation CSS framework was used to help style the app, including this page
  • The IDEs Visual Studio Code and IntelliJ IDEA was used to edit code
  • Node.js is used to host the St. Paul crime API

Interesting findings

  • Vue.js was confusing at first, but is very easy to use once you learn the basics and start getting comfortable it
  • V-If and V-For statements are incredibly useful
  • There are many more proactive police visits in the St. Paul area than we realized
  • Downtown is the smallest "neighborhood" by area, but had the highest number of incidents in nearly every category
  • The area around St. Thomas had very low rates of assault compared to the other neighborhoods on the map
  • The most common incidents around St. Thomas were Community Engagement Events and Discharging a Firearm within City Limits
